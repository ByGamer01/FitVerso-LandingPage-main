# FitVerso Landing Page

Landing page oficial de FitVerso - La app fitness gamificada con estilo anime.

## Sobre FitVerso

FitVerso transforma tu entrenamiento en una aventura epica. Gana XP, sube de nivel, completa misiones y conquista territorios con tu clan.

### Caracteristicas

- **Kira**: Tu companero gravitacional que multiplica tu XP segun tu racha
- **Batallas de Clanes**: Compite contra otros clanes en guerras semanales
- **Territorios**: Conquista y defiende territorios como en Clash of Clans
- **Sistema de Niveles**: Progresa desde E-Rank hasta S-Rank

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS
- TypeScript
